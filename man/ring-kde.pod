=head1 NAME

ring-kde - SIP and IAX2 compatible voice over IP softphone KDE client.

=head1 SYNOPSIS

ring-kde

=head1 DESCRIPTION

Ring-KDE is meant to be a robust enterprise-class desktop phone. It provides functions like call transfer, call hold, multiple lines, multiple accounts support.
Ring-KDE audio layer is build upon a native ALSA interface and and a native PulseAudio interface.
B<ring-kde> is a KDE client for Ring; it communicates with the core side through DBus. Ring package comes with the core, B<dring>.
